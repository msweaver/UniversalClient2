CONNECT 4 Open Source Platform for Maturing Interoperability During this demonstration, participants will see how organizations may leverage open source solutions to advance interoperability and implement bi-directional health data exchange. Leveraging all open source solutions - CONNECT, HIEOS and OpenEMPI - we demonstrate a cost-effective solution for implementing a disease-based registry. Then, we demonstrate how an organization, using the Nationwide Health Information Network services, may retrieve clinical data from and submit clinical data to the registry.
